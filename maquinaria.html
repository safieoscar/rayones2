Maquinaria y Equipos

Tejido Circular

Contamos con 2 tipos de máquinas para la elaboración del tejido circular: Single Knit y Doble Knit

Tejido Plano

Contamos con 3 tipos de máquinas para la elaboración del tejido plano: Telares de Pinza, Telares de Aire y Telares para toalla con Jackard.

Laboratorio

Contamos con un laboratorio completamente equipado, para la igualación de color, con espectrofotómetros certificados, programas para creación de recetas, dispensadores automáticos para químicos y colorantes, máquinas de teñir, foulard y Ramas para simular cualquier acabado requerido.

Tintorería

En nuestro proceso de teñido utilizamos colorantes dispersos, reactivos y VAT Dyes. Con estos últimos hemos diseñado procesos para alcanzar las mejores solideces del mercado. Se cuenta con dispensadores automáticos de químicos líquidos, sólidos y colorantes, lo que nos permite garantizar la reproducibilidad.

Estampado

Se tiene una maquina Rotativa de 12 colores con la que se puede trabajar con pigmentos, colorantes dispersos, reactivos y Vat Dyes. Todos dispensados de4 manera automática. Además, se cuenta con un vaporizador para el correcto desarrollo del color del estampado.

Acabados

Se cuenta con equipos para acabar el tejido circular en tubular o abierto según sea requerido, tenemos ramas para poder dar cualquier acabado a las telas, además de una sonorizadora para mejorar el performance y tacto. Y una cepilladora para poder ofrecer el acabado peach.

¡Más que vender telas, proveemos soluciones!
